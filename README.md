

# 🚀 NovaJobs – Job Portal Management System

## 📌 Project Overview

**NovaJobs** is a full-stack web-based Job Portal Management System designed to connect job seekers and employers on a single platform. It simplifies the recruitment process by providing a centralized system for job posting, job searching, and application tracking.


## 🎯 Objectives

* To build a centralized platform for job seekers and employers
* To automate the recruitment process
* To enable job search, application tracking, and job posting
* To provide secure login with role-based access


## ❗ Problem Statement

Traditional recruitment systems face several challenges:

* Manual and time-consuming hiring processes
* No centralized system for job applications
* Difficulty in tracking application status
* Poor communication between employers and job seekers
* Data mismanagement and duplicate applications

## 💡 Proposed Solution

NovaJobs provides:

* A web-based platform for job management
* Role-based access (Admin, Employer, Job Seeker)
* Real-time job updates and notifications
* Centralized database for efficient data handling
* Automated application tracking system

## 🧩 Modules

### 👤 Job Seeker Module

* Registration & Login
* Profile creation
* Resume upload
* Job search & filtering
* Apply for jobs
* Track application status

### 🏢 Employer Module

* Employer login
* Post job vacancies
* Manage job listings
* View applicants

### 📋 Job Management Module

* Add, update, delete job postings
* Filter and search jobs

### 📊 Application Management Module

* Track applications
* Update status (Shortlisted / Pending / Rejected)

### 🔔 Notification Module

* Email alerts for applications
* Status updates and job notifications

## 🔄 Workflow

1. User registers/logs in
2. Job seeker searches and applies for jobs
3. Employer posts jobs and manages applications
4. Admin monitors and updates application status
5. System sends notifications and updates
